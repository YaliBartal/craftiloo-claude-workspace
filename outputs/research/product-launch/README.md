# Product Launch Workspace

Complete product listing development outputs — listings + image plans + research.

## Structure

```
product-launch/
├── {product-slug}/
│   ├── listing/       # Amazon listing (title, bullets, backend keywords)
│   ├── image-plan/    # Image plan with exact copy for each image
│   ├── research/      # Shared research (competitors, keywords, Q&A, reviews)
│   └── README.md      # Product-specific summary
└── README.md          # This file
```
