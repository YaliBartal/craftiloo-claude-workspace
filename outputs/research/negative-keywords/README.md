# Negative Keywords

Proactive negative keyword lists generated from product knowledge and the Craftiloo catalogue.

## Folder Structure

```
negative-keywords/
├── briefs/     # Review lists + copy-paste output (what you read)
├── data/       # Archived search term exports (if provided as input)
└── README.md   # This file
```

## File Naming

| File Type | Format | Example |
|-----------|--------|---------|
| Review briefs | `{portfolio-slug}-negatives-YYYY-MM-DD.md` | `cross-girl-negatives-2026-02-19.md` |
| Data archives | `{portfolio-slug}-st-export-YYYY-MM-DD.xlsx` | `cross-girl-st-export-2026-02-19.xlsx` |

## What Goes Where

| Folder | Contents |
|--------|----------|
| `briefs/` | Full negative keyword review with categorized lists and copy-paste ready output |
| `data/` | Search term CSV/XLSX exports provided as optional input (archived after analysis) |

## How to Use

1. Run `/negative-keyword-generator` or say "generate negatives"
2. Pick a portfolio
3. Optionally provide a search term report
4. Review the categorized list
5. Approve and get copy-paste ready output
